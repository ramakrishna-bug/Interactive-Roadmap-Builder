# Interactive Roadmap Builder

An interactive web app that allows users to generate and track career learning roadmaps.

## 🚀 Features

- Select career path
- Generate roadmap dynamically
- Track progress with checkboxes
- Save progress using LocalStorage
- Reset progress anytime

## 🛠 Built With

- HTML
- CSS
- JavaScript
- LocalStorage API

## 📂 How to Run

1. Clone this repository
2. Open index.html in your browser
